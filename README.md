# 📚 Library Management System

A simple **Library Management System** built with **Django**, **HTML**, and **Bootstrap**. This app allows users to **manage books** efficiently by performing CRUD operations and viewing book details.  

![Python](https://img.shields.io/badge/Python-3.11-blue) ![Django](https://img.shields.io/badge/Django-4.2-green) ![Bootstrap](https://img.shields.io/badge/Bootstrap-5-purple) ![License](https://img.shields.io/badge/License-MIT-yellow)

---

## 🚀 Features

- ➕ Add new books to the library  
- 📋 View a list of all books  
- 🔍 View detailed information about each book  
- ✏️ Update existing book information  
- ❌ Delete books from the library  
- 📱 Responsive design using **Bootstrap**

---

## 🛠 Technologies Used

- **Backend:** Django  
- **Frontend:** HTML, CSS, Bootstrap  
- **Database:** SQLite (default with Django, can be changed)  

## 🎯 Usage

- Navigate through the app to **add, view, update, and delete books**.  
- Click on a book in the list to **see detailed information**.

✨ Made with ❤️ using Django and Bootstrap
